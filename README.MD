# 🛒 CodeAlpha_EcommerceStore

## 📌 Task: Simple E-commerce Store

### 📖 Description
A Full Stack E-commerce Web Application developed using **Node.js, Express.js, MongoDB, and EJS**.

The application allows users to register, login, browse products, add items to cart, and place orders.  
This project demonstrates practical implementation of backend routing, authentication system, database integration, and dynamic frontend rendering.

Built with **Node.js + Express.js + MongoDB + EJS (HTML, CSS, JavaScript)**.
